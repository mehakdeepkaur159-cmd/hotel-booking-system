## Features

### User Features

* User Registration and Login
* JWT Authentication
* Browse Available Rooms
* Search and Filter Rooms
* Book Rooms
* Cancel Bookings
* Payment Status Tracking
* Reviews and Ratings
* My Bookings Dashboard

### Admin Features

* Admin Dashboard
* Room Management (Add, Update, Delete)
* Booking Management
* User Management
* Revenue Analytics
* Booking Status Tracking

## Tech Stack

### Backend

* FastAPI
* PostgreSQL
* SQLAlchemy
* JWT Authentication

### Frontend

* HTML

* JavaScript
